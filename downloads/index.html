Straylight Simulation Tester

Straylight Simulation Tester can be used to test and FMU. You can install in on Windows only. It requires Java 1.7.
This version includes and FMU when you install it. If you would like to test a different FMU

  • Find the folder "C:\Program Files (x86)\Superb\StrayLight Simulation Tester\native_code\fmu"

  • rename it to 'fmu.bak'"

  • unzip your new FMU to the folder: C:\Program Files (x86)\Superb\StrayLight Simulation Tester\native_code\fmu

  • make sure that 'modelDescription.xml' is in the root of the folder
