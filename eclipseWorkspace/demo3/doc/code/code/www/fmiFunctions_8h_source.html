00004 /* This header file must be utilized when compiling a FMU.
00005    It defines all functions of Co-Simulation Interface.
00006    In order to have unique function names even if several FMUs
00007    are compiled together (e.g. for embedded systems), every "real" function name
00008    is constructed by prepending the function name by
00009    "MODEL_IDENTIFIER" + "_" where "MODEL_IDENTIFIER" is the short name
00010    of the model used as the name of the zip-file where the model is stored.
00011    Therefore, the typical usage is:
00012 
00013       #define MODEL_IDENTIFIER MyModel
00014       #include "fmiFunctions.h"
00015 
00016    As a result, a function that is defined as "fmiGetDerivatives" in this header file,
00017    is actually getting the name "MyModel_fmiGetDerivatives".
